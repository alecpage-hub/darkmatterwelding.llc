# Dark Matter Welding Website

A professional website for Dark Matter Welding, a welding and fabrication business.

## Project Overview

This is a static website designed to showcase Dark Matter Welding's services and projects to potential clients.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Service Showcase** - Display your welding and fabrication services
- **Portfolio Section** - Showcase completed projects
- **Contact Form** - Easy way for clients to get in touch
- **Smooth Animations** - Professional scroll animations and transitions

## File Structure

```
├── index.html      # Main HTML file
├── styles.css      # Styling and responsive layout
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## Getting Started

1. Open `index.html` in a web browser to view the site locally
2. Customize the content:
   - Update business name, services, and contact information
   - Add project images to the portfolio section
   - Modify colors and styling in `styles.css`

## Customization Tips

### Colors
- Primary color (orange): Change `#ffa500` in `styles.css`
- Dark background: Change `#1a1a1a` in `styles.css`

### Content
- Update services in the Services section of `index.html`
- Add portfolio images by replacing placeholder images
- Update contact information and form action

### Adding More Pages
- Create new `.html` files (e.g., `about.html`, `gallery.html`)
- Link them from the navigation in `index.html`
- Reuse `styles.css` for consistent styling
