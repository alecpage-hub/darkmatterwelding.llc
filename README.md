# Dark Matter Welding Website

A professional website for Dark Matter Welding, a mobile welding and fabrication business.

## Project Overview

Dark Matter Welding is a mobile welding and fabrication company founded by Alec Page and Martin Rivas. This is a professional website designed to showcase their services and reach potential clients across their service area.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Mobile-First Messaging** - Emphasizes our mobile service throughout the site
- **Service Showcase** - Highlights Custom Fabrication, Structural Welding, Mobile Welding, and Metal Art
- **Company Story** - About section tells the story of Alec, Martin, and their partnership
- **Professional Gallery** - Ready for project photos and testimonials
- **Contact Form & Direct Links** - Easy ways for clients to reach out
- **Social Media Integration** - Links to Instagram and Facebook
- **Purple/Magenta Theme** - Matches the Dark Matter Welding logo with glowing effects

## Customization

### Adding Project Photos
1. In the **Gallery** section, replace placeholder images with your project photos
2. Update project names and descriptions

### Changing Colors
All colors are defined in `styles.css`:
- Primary Magenta: `#d946ef`
- Primary Purple: `#a855f7`
- Background Black: `#000` or `#0a0a0a`

To change colors, find and replace these hex codes throughout the CSS.

### Adding Testimonials
Add a testimonials section after the Gallery section to showcase client feedback.

### Adding More Services
Duplicate a `service-card` div in the Services section to add more services.

## Deployment Options

### GitHub Pages (Free)
1. Go to repository settings
2. Scroll to "GitHub Pages"
3. Select "main" branch as source
4. Your site will be live at: `https://alecpage-hub.github.io/darkmatterwelding.llc/`

### Custom Domain
1. Point your domain (darkmatterwelding.llc) to GitHub Pages
2. Update repository settings with custom domain
